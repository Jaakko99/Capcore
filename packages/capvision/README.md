# recolor-plugin

Native on-device object segmentation and transformation for Capacitor apps.

## Install

To use npm

```bash
npm install @capcore/vision
````

To use yarn

```bash
yarn add @capcore/vision
```

Sync native files

```bash
npx cap sync
```
